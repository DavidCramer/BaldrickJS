No Coding

BaldrickJS reduces and can even elimiate writing javascript for ajax calls. Define ajax requests and desination via html data-* atributes.

Using a simple Request and Target concept, you simply set data-request="url" and data-target="#target" attributes in a trigger (an element to start the request).

This tells Baldrick to go fetch the url and place the result in the target.

Trigger

A trigger is an element that holds a request - target rule.
